# Foodie Flow

An internship-ready online food ordering system built in a `create-react-app` style structure.

## Unique Features

- AI-style food suggestion based on meal time
- Diet filters like high protein and low calorie
- Favorites and quick reorder interface
- Smart combo offer suggestions
- Live order tracker
- User review box with simple mood buttons
- Admin dashboard with revenue and order management
- Add food form and analytics section

## Team Split

- HTML/JSX structure: ready in `src/App.js`
- CSS styling: ready in `src/styles/app.css`
- Future React component/page split: folders and placeholder files are included

## Run

```bash
npm install
npm start
```
